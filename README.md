## 💼 Featured Engineering & ML Projects

### 🛍️ Retail Sales Forecasting & Inventory Optimization System
*A high-accuracy forecasting system integrated with a prescriptive decision-intelligence layer.*
- **Core Architecture:** Engineered statistical lag features and rolling averages to capture demand fluctuations. Constructed a multi-algorithm regression ensemble (XGBoost, LightGBM, Random Forest, KNN, SVR) for robust predictions.
- **Prescriptive Analytics:** Connected forecasting predictions to operations logic, automating optimal order quantity (EOQ) and safety stock recommendations to minimize carrying costs and mitigate stockouts.
- **Tech Stack:** `Python` · `Scikit-learn` · `Ensemble Methods` · `Feature Engineering`

---

### 🎯 CurveUrCareer — Dynamic Growth Ecosystem
*An automated cognitive assessment and academic navigation platform mapping student growth pathways.*
- **Platform Engineering:** Designed assessments evaluating student behaviors and academic skills. Built a parser analyzing capabilities and identifying skill gaps to produce dynamic learning roadmaps.
- **Data Stack:** Configured Supabase as a real-time data backplane handling relational mapping, user identities, and history states.
- **Tech Stack:** `React` · `Node.js` · `Express` · `Supabase` · `PostgreSQL`

---

### ⭐ REVU — Product Review Comparison Platform
*A full-stack aggregator displaying product ratings and reviews with comparative cross-platform analysis.*
- **System Design:** Engineered an asynchronous REST backend using FastAPI and structured review schemas inside a flexible MongoDB document store.
- **Features & UX:** Implemented JWT user verification. Crafted a React interface with dynamic search indexes and rating-based recommendation logic.
- **Tech Stack:** `React` · `FastAPI` · `MongoDB` · `JWT`
